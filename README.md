# Calculator Project 🧮

A functional, browser-based calculator built to demonstrate a solid grasp of JavaScript logic, event listeners, and CSS layout techniques.

## 🚀 [Live Preview](https://lydiahmbugua.github.io/calculator/)

---

## 🛠 Features
* **Arithmetic Operations:** Supports addition, subtraction, multiplication, and division.
* **Precision:** Handles decimal calculations accurately.
* **Smart UI:** 
    * "Clear" button to reset the state.
    * Display limits to prevent text overflow.
    * Error handling (e.g., preventing division by zero).
* **Responsive Design:** Optimized for both desktop and mobile screens.

## 💻 Tech Stack
* **HTML5:** Semantic structure for the calculator interface.
* **CSS3:** Custom styling using Flexbox/Grid for a clean, modern look.
* **JavaScript (ES6):** 
    * Complex logic for chaining multiple operations.
    * DOM manipulation for real-time display updates.
    * Event delegation for button handling.

## 🧠 Technical Challenges & Lessons
* **Managing State:** One of the biggest hurdles was managing the variables for the `firstNumber`, `operator`, and `secondNumber` to ensure the calculator remembers its current state during chained operations.
* **Edge Case Logic:** Implemented checks to prevent multiple decimal points in a single number and handled "equals" behavior when an operator hasn't been selected yet.
* **User Experience:** Focused on making the interface intuitive and ensuring the buttons feel responsive.
